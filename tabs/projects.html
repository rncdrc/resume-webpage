PROJECTS

  • PUPian the Explorer: A Python-Guided Navigation Game of Sintang Paaralan
    • The “PUPian the Explorer” is an interactive game application that is designed to immerse students in the campus life of the Polytechnic University of the Philippines (PUP).
  • Four-fect Triple: Identifying Groups of Values Forming a Pythagorean Triple Containing Multiples of Four
    • A study that aims to create a formula for Pythagorean triples in which the variable "a" is divisible by 4.
